Kids Learning Platform

Description:

  • Offers a vibrant platform designed to make education enjoyable for children.
  • Hosts a variety of engaging educational games aimed at enriching children's learning experiences.
  • Features an intuitive dashboard for effortless navigation.
  • Utilizes JWT authentication for personalized user dashboards.
  • Currently hosted on Vercel for easy accessibility.
  • Games feature accuracy and time spent metrics for performance evaluation.
  • Displays total time spent on the application.
  • Provides progress reports through visual representations for enhanced understanding of user performance and progress.
  • Showcases rapid development skills and a passion for innovative educational tools by Ashita.

Tech Stack:

  • MongoDB
  • ExpresJs
  • NodeJs
  • ReactJs
