TodoList App using MEAN

Description:

  • User registration and login using JWT
  • CRUD functionality for tasks (Create, Read, Update, Delete)
  • Task fields: title, description, due date, and status (pending, in progress, completed)
  • Task list view with filtering based on status
  • Pagination to limit the number of tasks displayed per page
  • Ability to mark tasks as completed
  • Backend API endpoints to support the frontend functionality
  • Error handling and validation on both frontend and backend
  • Deployment on Heroku

Tech Stack:

  • MongoDB
  • ExpresJs
  • NodeJs
  • Angular
