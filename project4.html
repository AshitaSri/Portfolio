Admin User Interface

Description:

  • Group Project
  • Implemented Admin and User authentication with OTP-based signup and forgot password functionality
  • Utilized JWT authentication for secure user authentication
  • Designed a hierarchical database structure: Head -> Mobiliser -> Admin -> User
  • Implemented data security measures at different levels to ensure secure data handling
  • Enabled user registration using Aadhar card and PAN card, with data storage on Firebase
  • Utilized data visualization techniques and automated payment updates based on user hierarchy

Tech Stack:

  • MongoDB
  • ExpresJs
  • NodeJs
  • React
