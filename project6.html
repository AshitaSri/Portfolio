E-commerce Website with React

Description:

  • e-Commerce website using ReactJS
  • Features include: Trending products display, offer banners, newsletter subscription form
  • Product pages for men's, women's, and kids' categories
  • Single product page with image gallery, price, description, and related products list
  • Shopping cart page to display added products
  • Login and register pages for user authentication

Tech Stack:

  • React
